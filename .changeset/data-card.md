Add the `DataCard` kit — a small grammar for building compact data cards. Compose a `DataCard` shell with the `DataCardTopline`, `DataCardBigNumber`, `DataCardLabel`, `DataCardColumns`, `DataCardColumn`, `DataCardChartWrap` and `DataCardFootnote` primitives to pair a headline statistic with supporting labels, charts and footnotes. Each card is its own CSS size container, so its type scales to the card's own rendered width. `DataCardRow` lays several cards out side by side and turns into a horizontal swipe carousel (with scroll-chaining guards) only when the cards don't all fit.
