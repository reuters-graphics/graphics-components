# LineChart Component

A flexible, responsive line chart component built with D3.js. Supports single and multiple lines, small multiples, and custom axis formatting.

## Features

- **Single & multiple lines**: Display one or many data series on a single chart
- **Small multiples**: Lay out charts in a grid for easy comparison
- **Responsive**: Automatically adapts to container width with customisable breakpoints
- **Y and X-axis formatting**: Flexible formatting with units or custom functions
- **Date format support**: The component automatically detects and conve
 rts date strings to `Date` objects. You can pass dates as ISO strings, date strings, or `Date` objects directly—the component handles them all seamlessly.
- **X-axis date formatting**: Configure date labels, auto formatted in the Reuters style by default
- **Custom snippets**: Optionally, add custom snippets on top of the SVG container
- **Legend support**: Automatic legend generation with customisable colours
- **Auto-detection of dates**: X-axis date values are automatically detected and converted from strings (ISO format, date strings) without extra configuration
