# Axis formatting

Axis label formatting are customised via the `xAxisConfig` and `yAxisConfig` props. Both props accept an object with multiple formatting options.

> 💡TIP: If labels are cut off, adjust the `margin` prop to provide more space for the labels. By default, it is set to `{top: 20, bottom: 20, left: 20, right: 20}`.


## X-axis formatters

**`xAxisConfig.xAxisDateFormat`**

  Default option. Applies a simple [time format pattern](https://www.strfti.me/), such as `"%b '%y"` for `Jan. '24`. Defaults to `'%b %d, %Y'` (`Jan. 1, 2024`).

```svelte
<LineChart 
  {data}
  xKey="date"
  yKey="endDayVal"
  xAxisConfig={{ xAxisDateFormat: "%b '%y" }}
 />
```

**`xAxisConfig.xFormatter`**

Custom JavaScript function. Provides full control over label formatting and the ability to return multi-line labels. 

Overrides `xAxisDateFormat`.

```svelte
<script>
import { LineChart, prettifyDate } from '@reuters-graphics/graphics-components';

// Custom function to conditionally format x axis labels
const xFormatterShort = (d: Date, idx: number) => {
  // Get the abbreviated month name e.g. "Jan"
  const month = d.toLocaleDateString('en-US', { month: 'short' });
  
  // Get the full year e.g. 2024
  const fullYear = d.getFullYear();
  
  // Get the last 2 digits of the year e.g. "24"
  const shortYear = String(fullYear).slice(-2);

  // For the first tick show the full year e.g. "Jan 2024"
  // For all other ticks show the short year with apostrophe e.g. "Jan '24"
  const dateLabel = idx === 0
    ? `${month} ${fullYear}`
    : `${month} '${shortYear}`;

  // Format the date in Reuters style
  return prettifyDate(dateLabel);
};
</script>
<LineChart 
  {data}
  xKey="date"
  yKey="endDayVal"
  xAxisConfig={{ xFormatter: xFormatterShort }}
 />
```

### Multi-line axis labels

To add multi-line axis labels, pass a function that returns an array of values into `xFormatter`:

```svelte
<script>
import { LineChart, prettifyDate } from '@reuters-graphics/graphics-components';

// Custom function to format x axis labels as multi-line labels
  const xFormatterTwoLine = (d: Date): string | string[] => {
    const month = d.toLocaleDateString('en-GB', { month: 'short' });
    const year = d.getFullYear()

    // Return an array to create multi-line labels. Each item in the array will be rendered on a new line.
    // Use `prettifyDate()` to format the month in the Reuters style.
    return [prettifyDate(month), year];
  };
</script>

<!-- Adjust margin if labels are cut off -->
<LineChart 
  {data}
  xKey="date"
  yKey="endDayVal"
  xAxisConfig={{ xFormatter: xFormatterTwoLine }}
  margin: { left: 15, right: 60, bottom: 40 },
 />
```
