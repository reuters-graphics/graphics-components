# SiteFooter

The `SiteFooter` component contains a site footer with graphics referrals styled for the Reuters dotcom site, ported from [Raptor UI components](https://github.com/tr/rcom-arc_raptor-ui/tree/develop/packages/rcom-raptor-ui_common/src/components/site-footer).

NOTE THIS IN THE PR:

> **Note:** In the Graphics Kit, you can find this component in `pages/+page.svelte`. Customise it there for the default page.

```svelte
<script>
  import { SiteFooter } from '@reuters-graphics/graphics-components';
</script>

<SiteFooter />
```

<Canvas of={SiteFooterStories.Demo} />

---

Colours are customised by the [`Theme`](?path=/docs/theming-theme--default) component.

```svelte
<script>
  import { SiteFooter, Theme } from '@reuters-graphics/graphics-components';
</script>

<Theme base="dark">
  <SiteFooter />
</Theme>
```

---

Remove graphics referrals.

```svelte
<script>
  import { SiteFooter } from '@reuters-graphics/graphics-components';
</script>

<SiteFooter includeReferrals={false} />
```
