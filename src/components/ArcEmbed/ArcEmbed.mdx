# ArcEmbed

`ArcEmbed` is a small, domain-free starter kit for building iframed embeds for the **Arc** content management system that runs Reuters.com. It supplies the shared chrome — a light `Theme`, the Reuters Knowledge font, [Pym.js](https://blog.apps.npr.org/pym.js/) iframe resizing and a simple stacked layout — so any graphic can drop its own header, stage (the main visual) and footer into named snippets and match the look of the homepage.

The kit has three pieces:

- **`ArcEmbed`** — the frame: header, a positioned `stage` for your visual, and an optional footer.
- **`ArcHeader`** — a homepage-style title block with an optional kicker, headline and dek.
- **`ArcFonts`** — loads the Reuters Knowledge web font. Render it once near the root of your embed.

```svelte
<script>
  import {
    ArcEmbed,
    ArcHeader,
    ArcFonts,
  } from '@reuters-graphics/graphics-components';
</script>

<ArcFonts />
<ArcEmbed stageHeight={420}>
  {#snippet header()}
    <ArcHeader
      kicker="Wildlife"
      kickerUrl="https://www.reuters.com/"
      headline="The great white's comeback"
      dek="Conservation efforts have helped a feared predator rebound along the coast."
    />
  {/snippet}
  {#snippet stage()}
    <!-- Your map, chart or photo goes here. -->
    <img src={photo} alt="A great white shark" />
  {/snippet}
  {#snippet footer()}
    <p class="caption">A great white shark off the coast. REUTERS</p>
  {/snippet}
</ArcEmbed>
```

## The stage

The `stage` is a positioned (`position: relative`) area sized by the `stageHeight` prop, so your graphic and any controls or overlays can absolutely position themselves within it. Anything that needs to sit _below_ the visual — captions, summaries, credits — goes in the optional `footer` snippet.

```svelte
<ArcEmbed stageHeight={300}>
  {#snippet header()}
    <ArcHeader headline="A simple embed" />
  {/snippet}
  {#snippet stage()}
    <img src={photo} alt="A great white shark" />
  {/snippet}
</ArcEmbed>
```

## Fonts

`ArcFonts` registers the Knowledge typeface under the `ArcKnowledge` family so it never collides with any `Knowledge` font already present on the host page. The font files are served from the Reuters graphics CDN, which sends the cross-origin header browsers require for `@font-face`. Render `ArcFonts` once near the root of your embed layout.
