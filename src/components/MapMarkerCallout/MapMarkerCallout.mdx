# MapMarkerCallout

A small, presentational callout for annotated map points. It renders a framed
surface, a leader line and an anchor dot, but it does **not** know about map
libraries, coordinates, overlay lifecycles, click handling or project-specific
data.

Use it inside whatever overlay system positions an element at a projected map
point.
